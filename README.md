# Kioskopago

## Requisitos

- Node.js 18+
- yarn

## Instalación

```bash
# Clonar el repositorio
git clone git@github.com:API-DEV-LAB/kioskopago-web.git

# Instalar dependencias
yarn install
```

## Variables de Entorno

Crear un archivo `.env.local` con las siguientes variables:

Ambiente de desarrollo

```bash
NEXT_PUBLIC_API_URL=
```

## Comandos

```bash
# Desarrollo
yarn dev

# Build producción
yarn build

# Iniciar producción
yarn start

# Formatear código
yarn format
```
